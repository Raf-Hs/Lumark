# Installation & Setup

### Windows

1. Download the latest installer `.exe` package from [GitHub Releases](https://github.com/AlbertArakelyan/Lumark/releases/latest)
2. Open the downloaded file
3. Follow the installation wizard
4. Open the app after installation

### Mac

1. Download the latest installer `.dmg` package from [GitHub Releases](https://github.com/AlbertArakelyan/Lumark/releases/latest)
2. Open the downloaded file
3. Open DMG and drag Lumark.app to Applications
4. Find the app in Applications and open it

### Linux

1. Download the latest installer `.deb` package from [GitHub Releases](https://github.com/AlbertArakelyan/Lumark/releases/latest)
2. Open the downloaded file in the file manager
3. If a security warning appears, click "Run Anyway"
4. Follow the installation wizard
5. Open the app after installation
